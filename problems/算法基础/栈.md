# 栈
## 基本操作
* 性质:先进后出
* 定义:stack<int> s;
* 输入:s.push();
* 删除:s.pop();
* 输出：cout<<s.top();
## 模拟栈
1. 模拟
```c++
    //https://www.acwing.com/problem/content/830/
    #include <iostream>
    using namespace std;
    const int N=1e5+10;
    int m;
    int stk[N],tt;//tt是长度
    int main(){
        cin>>m;
        while(m--){
            string op;
            int x;
            cin>>op;
            if(op=="push"){
                cin>>x;
                stk[++tt]=x;
            }
            else if(op=="pop") tt--;
            else if(op=="empty") cout<<(tt?"NO":"YES")<<endl;
            else cout<<stk[tt]<<endl;
        }
        return 0;
    }
```

## 例题
1. 
* 题目概括：给一组数，按照原顺序入栈，尾巴可随时多次出栈，看能不能排成询问的顺序
* 样例：
```c++
5
1 2 3 4 5
2
3 2 1 5 4 YES
3 4 2 1 5 YES
```
* 思路：3输入，读取数组直到3，因为接下来不是2out就是4out，所以先退一位看看是不是2，如果不是先进一位空位再读4，再比较，再退一位，以此类推；
```C++
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <malloc.h>

int main()
{
    int n, i, t;
    int s[10010];
    int a[10010];
    scanf("%d", &n);//length
    for(i = 0; i < n; i++) scanf("%d", &a[i]);
    int p, q;
    int m;
    scanf("%d", &m);//test number
    while(m--)
    {
        p = 0;//长度
        q = -1;//初始化下标
        for(i = 0; i < n; i++)//
        {                    //依次输入比较
            scanf("%d", &t);//
            while(q == -1 || s[q] != t)//相等就跳过，下面q--删除
            {
                if(p >= n) break;
                s[++q] = a[p++];//co
                //先++，因为要判断，若后++判断全为true
            }
            //如果有后面的符合顺序，则逆序使q--(双向，退一位)
            if(t == s[q])
                q--;
        }
        if(q == -1)//全输出，q=-1;
            printf("yes\n");
        else
            printf("no\n");
    }
    return 0;
}
```
2. 
* https://www.acwing.com/problem/content/3305/
* 题目概括：给式子计算出值
* 样例：(2+2)*(1+1)。输出8
### 思路
* 两栈，一个操作栈，一个结果栈；
* 每次录入操作时先和栈顶比较优先级，'+' < '*',不能相等，若输入的高则直接录入，不然要计算出值后录入结果栈，直到栈顶此时小于录入的或者操作栈空，此时结果栈只有一个结果;
* 比较优先级用unordered_map捆版表；
* 遇到左括号录入，遇到右括号计算直到遇到左括号；
```c++
#include <iostream>
#include <stack>
#include <string>
#include <unordered_map>
using namespace std;

stack<int> num;
stack<char> op;

//优先级表，将符号和等级关联
unordered_map<char, int> h{ {'+', 1}, {'-', 1}, {'*',2}, {'/', 2} };


void eval()//求值
{
    int a = num.top();//第二个操作数
    num.pop();

    int b = num.top();//第一个操作数
    num.pop();

    char p = op.top();//运算符
    op.pop();

    int r = 0;//结果 

    //计算结果
    if (p == '+') r = b + a;
    if (p == '-') r = b - a;
    if (p == '*') r = b * a;
    if (p == '/') r = b / a;

    num.push(r);//结果入栈
}

int main()
{
    string s;//读入表达式
    cin >> s;

    for (int i = 0; i < s.size(); i++)
    {
        if (isdigit(s[i]))//数字入栈//isdigit判断是否是数字
        {
            int x = 0, j = i;//计算数字
            while (j < s.size() && isdigit(s[j]))//多位数
            {
                x = x * 10 + s[j] - '0';
                j++;
            }
            num.push(x);//数字入栈
            i = j - 1;//自身带i++
        }
        //左括号无优先级，直接入栈
        else if (s[i] == '(')//左括号入栈
        {
            op.push(s[i]);
        }
        //括号特殊，遇到左括号直接入栈，遇到右括号计算括号里面的
        else if (s[i] == ')')//右括号
        {
            while(op.top() != '(')//一直计算到左括号
                eval();
            op.pop();//左括号出栈，有括号没入
        }
        else
        {//unordered_map自动查找
            while (op.size() && h[op.top()] >= h[s[i]])//待入栈运算符优先级低，则先计算
                eval();//可能有多个*or/
            op.push(s[i]);//操作符入栈
        }
    }
    while (op.size()) eval();//若操作栈还有剩余的则进行计算
    cout << num.top() << endl;//输出结果栈结果
    return 0;
}
```
